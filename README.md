# CoolMessage
Exemplo de uso do RabbitMQ com Docker

# Utilização
Para rodar a aplicação no docker, digite o comando abaixo na raiz do projeto
```bash
docker-compose up
```
Para acessar o RabbitMQ 
http://localhost:15672

Para acessar o swagger da aplicação
https://localhost:5001/swagger/index.html

# Dependências
* [Docker](https://www.docker.com/)
* [.Net Core](https://dotnet.microsoft.com/en-us/download)
